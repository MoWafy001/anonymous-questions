# Env Variables

* PGDATABASE
* PGHOST
* PGPASSWORD
* PGPORT ( usually `5432` )
* PGUSER
* SERVEREMAIL ( `must be a Gmail`, unless you modify the code )
* SERVEREMAILPASSWORD
### *These two may be needed if a secure connection is required*
* NODE_TLS_REJECT_UNAUTHORIZED=`'0'`
* PGSSLMODE=`"require"`
